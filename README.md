# chess-repertoire-builder
Build chess opening repertoires.

The chess piece artwork used in this program was created by Maurizio Monge and is used with permission, requiring attribution. Source: https://poisson.phc.dm.unipi.it/~monge/chess_art.php

The program uses an opening api from LiChess https://lichess.org/api

You can try this program at https://restframe.us/openingstudy/
